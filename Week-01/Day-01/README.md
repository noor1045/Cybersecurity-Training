## Kali Linux Installation

For this task, I installed Kali Linux in Oracle VirtualBox.
First, I created a new virtual machine and attached the Kali Linux ISO file. I assigned 4 GB of RAM, 2 processor cores, and approximately 26.8 GB of virtual disk space.
I used the graphical installation option and completed the basic settings, including language, hostname, username, password, and time zone.
For disk partitioning, I selected the guided option and used the entire virtual disk. I installed the Xfce desktop environment along with the recommended Kali Linux tools.
During the installation, the GRUB boot loader initially failed. I repeated the GRUB installation process and selected the VirtualBox hard disk `/dev/sda`. After that, GRUB was installed successfully.
The virtual machine restarted successfully, and I was able to log in to the Kali Linux desktop.
## Virtual Machine Configuration
- Platform: Oracle VirtualBox
- Operating System: Kali Linux 2025.3
- RAM: 4 GB
- Processors: 2 CPU cores
- Virtual Disk: 26.8 GB
- Desktop Environment: Xfce
- Network Mode: NAT
